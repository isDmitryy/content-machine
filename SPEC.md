Создай одностраничное демо-приложение на Next.js + Tailwind CSS.

Концепция: одна сырая идея превращается в полноценную медиасистему.

Без бэкенда, без базы данных, без API. Используй клиентскую mock-генерацию на основе текста, который ввёл пользователь.

Стиль:
тёмный графитовый фон, фиолетовые и электрически-синие акценты, ощущение премиального AI-операторского дашборда. Серьёзно, остро, не игрушечно.

Сценарий:
Пользователь вводит идею.
Нажимает кнопку: «Развернуть в контент-систему».
Покажи короткое анимированное состояние генерации.
Затем карточки появляются одна за другой.

Сгенерируй:
структуру YouTube-видео;
Telegram-пост;
5 хуков для Reels / Shorts;

Добавь визуальный пайплайн:
Идея → Нарратив → Форматы → Дистрибуция

У каждой карточки должны быть:
название формата, сгенерированный текст, кнопка копирования, leverage score, плавная glow-анимация.

Цель приложения:
сделать fast-tech-прототип, который за несколько секунд показывает новую AI-era способность:
одна сильная мысль может превращаться в целую контент-машину.
